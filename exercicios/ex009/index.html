ol significa Ordered List (Listas Ordenadas)

il, significa list iten (item de lista)

O programa para criar lista nos dá 3 opções de marcadores:

  1. Marcadores Numéricos: 1, 2, 3, 4, 5, ...
  2. Marcadores Alfabéticos: A, a, B, b, C, c, D, d, ...
  3. Marcadores Romanos: I, i, II, ii, III, iii, IV, iv, ...

Para utilizar estes marcadores digita-se dentro da tag de abertura ol a css type e dê enter, aparecendo a área para digitar a sua opção.

O termo type indica o tipo de marcador que se deseja usar na organização de uma lista. Todos estes tipos irão começar do primeiro caracter do tipo numérico, partindo do número 1, tipo alfabético, partindo sempre da letra A, a; e do tipo românico partindo sempre do I, i.

Caso se deseje que uma lista inicía-se de um outro ponto de partida e não dos caracteres iniciais, deve-se acrescentar ao type a tag start que significa de que ponto você quer que comece a organizar sua lista. Na tag start não se usa caractere alfabético e nem romano, usa-se somente caractere numérico. O número a ser usado deve ser referente a posição que o caractere ocupa em sua estrutura. Exemplo a letra e ocupa a posição 5 dentro do alfabeto. A letra j ocupa a décima posição dentro do alfabeto. Então se for iniciar uma lista a partir da letra jota, o código deve ser escrito da seguinte maneira:

ol type="A" start= "10". o start 10 define que a lista começará da letra j. Vamos ver alguns exemplos?

Listas Não Ordenadas (ul)

ul significa unordered list, ou listas não ordenadas. Este tipo de lista segue o mesmo princípio das listas ordenadas, no entanto, seus marcadores são de três tipos apenas: quadrado (square), círculo aberto (circle), e um círculo aberto (disk).

Lista de Definições (dl, dt, dd)

A lista de definição (dl) funciona como a lista de um dicionário, onde você tem uma lista de titulo (dt) e abaixo de cada título você tem a lista de descrição (dd) informando o significado da palavra do título.
