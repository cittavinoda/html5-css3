Semântica

No HTML5 as tags tem como função dar um significado ao texto e não a forma do texto. Exemplo: Ao sublinhar um texto com a tag u para dar um destaque ao endereço de uma empresa, ou mesmo ao seu endereço, este comportamento é referente a forma, isto é, o texto foi sublinhado para chamar a atenção do leitor para o fato de le ser um endereço. Entretanto, esta tag é apenas uma forma de destaque. Se o mes mo texto for escrito dentro de uma tag html semântica como por exemplo a tag adress, ou endereço, esta tag além de dar um destaque ao endereço, permitirar ao clicar nele o direcionamento para o Google Map, mostrando a localização do lugar endereçado.
