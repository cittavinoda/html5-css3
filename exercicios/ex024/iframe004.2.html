Bloqueando Ações Maliciosas

Para bloquear qualquer ação maliciosa que possa estar contido no site inserido no seu iframe usa-se o método "sandbox" no padrão "sandbox". Este método produz segurança máxima para o seu iframe e não permite que o usuário tenha acesso direto aos eventos de solicitação de dados, de ativação de script, acesso a formulários, etc.

Sandbox é um método, e este método possui vários padrões de atuação. O padrão sandbox bloqueia todo acesso do usuário do seu site aos arquivos que estejam dentro do iframe. A colocação de um iframe dentro do seu site, abre uma porta de entrada para usuários e outros sites. Há sites ocultos que deixam um formulário, ou outro elemento de entrada de dados com a intensão de capturar dados dos usuários que entram no seu site, e podem causar perturbações ao usuário que acessa o seu site. Este evento não é bom, porque o seu site fica marcado como um site problemático. No arquivo iframe004.3 ver-se-á como manter o seu iframe bloqueado com uma segurança extra.
