Meu Site Principal

Nesta página do nosso site criaremos um iframe e dentro dele vamos inserir um site externo. Neste site externo encontra-se uma aba para que o usuário do nosso site possa cadastrar seus dados pessoais. Esta opção de cadastro, dentro do site externo, pode ser uma artimanha para pegar dados dos usuários que entram em nosso site, por empresas de propaganda, que enviam para o sistema do usuário diversos tipos de propaganda, ou mesmo, sites que pegam informações do usuário para coisas suspeitas.

Obs: Este formulário representa um site externo e que foi inserido neste site por meio de um iframe. Se você tem um site externo dentro do seu site por meio de um iframe, você deve entender que o iframe é uma porta de entrada para qualquer usuário, ou para qualquer site ter acesso aos dados do seu usuário, e isto caracteriza um grande problema. No iframe004.2.html, você verá como este problema poderá ser resolvido facilmente com um método que bloqueia o preenchimento de dados no formulário, dado como suapeito no site inserido em seu iframe.
