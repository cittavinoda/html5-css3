Minha Página

Na página 5 (pasg005.html) foi criado um script com um botão de acesso, e tendo como saída a confirmação de botão clicado "Você clicou!". O iframe, através do "src" acessa a pasta "páginas-extras" e a página pag005.html do seu site. Desta forma temos uma porta de entrada aberta pasra qualquer um que queira entrar.

Nesta configuração, o usuário terá acesso ao script que está dentro site que você colocou em seu iframe. Se o script for maliciosso, de alguém mal intensionado, você estará dando permissão para que este script maliciosso colete dados importantes do seu usuário, e isto não é nada bom. No iframe005.2 vermos como bloquear todo o iframe para que nada dentro do site que está dentro do seu iframe possa acessar as informações do usuário.
