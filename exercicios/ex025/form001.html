Todo formulário tem que estar dentro da tag "form".

Toda chamada de controle, por exemplo, a solicitação para digitar o nome, Nome: deve ser seguido da caixa de controle que é criado pelo parâmetro input. Como o input é um texto, o parâmetro a ser selecionado é input:text.

Para o envio da informação deve-se criar um botão com a ação de enviar. Para criar este botão usa-se o parâmetro input:submit. Juntamente com o input:submit, vem o parâmetro value, que é o elemento usado para dar nome ao botão submit.
