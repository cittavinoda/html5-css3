Checkbox
O <input type="checkbox"> define uma caixa de seleção.

Radio Button
O <input type="radio"> define um botão de opção.

Radio Button (Botão de Radio) Os botões de opção são normalmente apresentados em grupos de rádio (uma coleção de botões de opção que descrevem um conjunto de opções relacionadas). Apenas um botão de opção em um grupo pode ser selecionado ao mesmo tempo.

Nota 1: O grupo de rádio deve ter compartilhado o mesmo nome (o valor do atributo name) para ser tratado como um grupo, entretanto, o id de cada elemento deve ser diferente. Depois que o grupo de opções é criado, selecionar qualquer botão de opção nesse grupo desmarca automaticamente qualquer outro botão de opção selecionado no mesmo grupo.
Para vários elementos pertencerem ao mesmo grupo, eles tem que ter mesmo "name" e "id's" diferentes.
Você pode ter quantos grupos de rádio quiser em uma página, desde que cada grupo tenha seu próprio nome.
Cada grupo tem que ter seu próprio nome, mas todos os elementos de cada grupo tem que ter mesmos names e id's diferentes.

Nota 2: O atributo value define o valor exclusivo associado a cada botão de opção. O valor não é mostrado ao usuário, mas é o valor que é enviado ao servidor em "enviar" para identificar qual botão de opção foi selecionado.

Atributo Checked.
O atributo checked é um atributo booleano. Quando presente, ele especifica que um <input> elemento, deve ser pré-selecionado (marcado) quando a página é carregada.
O atributo checked pode ser usado com <input type="checkbox"> () e <input type="radio"> ().
O atributo checked também pode ser definido após o carregamento da página, com um JavaScript.
No fieldset que for usado o elemento input radio, não é permitido o uso de mais do que um atributo checked. No fieldset que for usado o elemento input checkbox, pode-se usar o atributo checked sem nenhuma restição.
