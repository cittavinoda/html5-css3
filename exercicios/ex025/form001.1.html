Meu Primeiro Formulário

Para onde são enviado os dados?

  • Ao clicar no botão enviar, os dados são enviados para um servidor, que pode ser Java, JavaScript, php, etc. Neste caso o servidor escolhido será o php.
  • Para selecionar o servidor php deve-se ativar no "form" o parâmetro action="" e dentro de suas aspas deve-se escrever o nome do arquivo de servidor que receberá as informações enviadas."
  • Para remover a caixa de menu de opções quando se passar o mause sobre a caixa de controle, deve-se inserir no "form" o parâmetro autocomplete="off".
  • Para ligar o termo Nome: ou Sobrenome a sua caixa de controle utilizamos o parâmetro "lable". Os "lables" faz com que o termo Nome: ou Sobrenome seja a etiqueta de identificação de suas caixas de controle.
  • Para o HTML e o PHP é mais importante o objeto "name" do que o objeto "id", e para o JavaScript e o Lable é mais importante o objeto "id".
  • Para interligar o termo a suas caixas de controle os termos "Nome" e "Sobrenome" devem ser envelopados com a tag "lable". A tag lable oferta apenas o parâmetro (for=" "). O parâmetro for deve receber como valor, o valor declarado nas "id's".
