Comando pre

O comando "code" muito embora padroniza a largura das letras tornando o texto mais visível, para o caso de padronizar a larguras dos caracteres em um código, de python por exemplo, ele não organiza o texto editado na forma de código, sendo necessário o envelopamento do código com o code pela tag pre.

Observe a diferença nos códigos escritos abaixo:

Primeiro Código envelopado apenas com a tag code

num = int(input('Digite um número')) if num % 2 == 0: print('O número {num} é PAR') else: print('O número {num} é ÍMPAR') print('Fim do programa')

Observe que dessa forma o código será escrito na mesma linha.

Avançando ou Recuando um Código Dentro do Bloco

Pra você poder avançar um código dentro do bloco, selecione o código e zaperte a tecla tab. A cada aperto datecdla tab o código anda um pixel para a direita.

Para recuar um código dentro do bloco, selecione o código, aperte a tecla shift e tab, A cada aperto da tecla tab com o shift pressionado o código irá recuar um pixel para a esquerda.
