Usando Links Externos

Você pode acessar o meu repositório público no GitHub.

Para acessar o repositório GitHub, usa-se a tag a de âncora. A âncora pede que incorpore a ela a URL do site que se deseja visitar ao clicar na palavra que estiver ancorado tal URL. Como exemplo usaremos o texto "repositório no GitHub". Este texto será envelopado com a tag a.

Você pode acessar o meu repositório público no GitHub.

O que temos aqui? O comando a href faz uma referência a hipertexto que torna o texto envelopado em um link que dá acesso ao site no github.
