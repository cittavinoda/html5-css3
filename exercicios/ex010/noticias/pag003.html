Voltar para a primeira página

A página pag003 foi criada dentro da pasta noticias. A primeira página (index.html) e a segunda página (pag002.html) não estão dentro da página notícia. Se você tentar referenciar esta página de dentro da notícias, vai dar erro, porque o html irá procurar o comando index.html dentro da página notícias. Para evitar este erro devemos informar ao html que temos que procurar o index.html em outra página, e, para isso, usa-se o comando (../). Este comando informará em que pasta o index.html, referente a primeira página se encontra.
