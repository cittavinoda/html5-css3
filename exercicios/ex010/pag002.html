Vamos para a terceira página?

A página pag003 foi criada dentro da pasta noticias. A primeira página (index.html) e a segunda página (pag002.html) não estão dentro da página notícias. Se você tentar referenciar esta página de dentro da notícias, vai dar erro, porque o html irá procurar o comando pag003.html fora da página notícias. Para evitar este erro devemos informar ao html que temos que procurar o pag003.html na página notícias, e, para isso, usa-se o comando Ctrl+espaço. Este comando informará em que pasta o pag002.html, referente a onde a segunda página se encontra.
