Desafio das Mensagens

Você pode escreverum parágrafo de qualquer jeito, basta colocar tudo entre o par de tags p e /p.

Se precisar quebrar o texto em algum lugar específico, basta usar a tag br (<br>)

Para representar uma tag dentro dos sinais maior (>) e menor (<), deve-se fazer o uso dos seguintes códigos & lt para o sinal de maior e & gt para o sinal de menor.

Observação:
Para que os símbolos maior e menor se manifestem na forma de símbolos no texto o código (& lt) e (& gt) não pode ter espaço entre eles. < maior - menor>.
